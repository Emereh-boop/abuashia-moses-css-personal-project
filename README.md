# Personal Portfolio Website
**Developer:** Abuashia Moses Etsorzi  
**Project:** CSS Personal Project 1 - Turing Tech LLC  

A professional, responsive single-page portfolio built using semantic HTML5 and modern CSS (Flexbox & Grid).

## 🚀 Features
- **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop.
- **Sticky Navigation**: Easy access to sections at all times.
- **Modern Layouts**: Built using CSS Grid for the Skills and Projects sections.
- **Smooth Scrolling**: Enhanced user experience for in-page navigation.

## 📁 Project Structure
- `index.html`: Main semantic HTML5 structure.
- `css/styles.css`: Custom styling including variables and media queries.
- `images/`: High-quality assets and project screenshots.

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Custom Variables)
- Google Fonts (Poppins & Roboto)

## 📋 How to Run
1. Clone or download the repository.
2. Ensure your images are placed in the `/images` folder as `profile.jpg` and `project1.jpg`.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
